#Install
## APP (Angular)
 - bower install
## NODE (4.0)
 - cd app/sources/nodejs
 - npm install
## NODE CHAT
 - cd app/sources/nodejs
 - npm install
 - node socket.js
 - http://localhost:3000/
## NODE CHAT V2 (https://github.com/socketio/socket.io/tree/master/examples/chat)
 - cd app/sources/nodejschat
 - npm install
 - node index.js
 - http://localhost:3000/
